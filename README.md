# Clarity

Clarity is a progressive method for improving project planning docs so
collaborators can rely on precise, consistent context.

## This Repo

- `methodology/` contains the Clarity artifacts (CLARITY, flows, labels, steps).
- `setup/` contains setup instructions for installing Clarity in a target repo.
- `scripts/` contains a helper to copy the methodology into `.clarity/`.

## Use in another repo

See `setup/clarity-setup.md` for copy/install steps and the AGENTS snippet.
