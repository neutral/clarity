# Step: improve-flow

Purpose: Make docs read in a logical order.

Inputs:
- Target set (paths/headings/labels)
- Audience expectations

Steps:
1) Reorder sections from context to details.
2) Group related concepts and remove duplicates.
3) Add short transitions where context shifts.
4) Ensure headings still match the intended narrative.

Outputs:
- Clearer structure and scannability.
