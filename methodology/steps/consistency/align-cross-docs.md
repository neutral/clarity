# Step: align-cross-docs

Purpose: Align overlapping topics across files to avoid drift and duplication.

Inputs:
- Target set (paths/headings/labels)

Steps:
1) Identify overlapping topics or repeated sections.
2) Choose a canonical source for each topic.
3) Reconcile discrepancies or replace duplicates with references.

Outputs:
- Cross-document alignment and reduced duplication.
