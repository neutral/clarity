# Step: fill-gaps

Purpose: Add missing information required for clarity or decision-making.

Inputs:
- Target set (paths/headings/labels)
- Open questions or known gaps

Steps:
1) Mark missing information and its impact.
2) Add content if known, or add labeled placeholders (Question/Assumption).
3) Ensure gaps are captured in the run report.

Outputs:
- Completed sections or explicit placeholders for missing info.
